# Wishlist App

This is a wishlist application where you can add links from any website to track prices, details, and other information about the products you want.

## Features
- Add product links from any website
- Automatically see the product name, price, and details
- Organize your wishlist easily
- View and track price changes

## How it works
1. Paste a product link into your wishlist
2. The app automatically fetches the main product information (name, price, image, etc.)
3. Track all your wishes in one place

## Suggested technologies
- Backend: Node.js, Python, or any language of your choice
- Frontend: React, Vue, Angular, or similar
- Web scraping: Puppeteer, Playwright, BeautifulSoup, etc.

## Possible improvements
- Price drop notifications
- Wishlist sharing
- User login

# Wishlist App

Este é um aplicativo de wishlist onde você pode adicionar links de qualquer site para acompanhar preços, detalhes e outras informações dos produtos que deseja.

## Funcionalidades
- Adicione links de produtos de qualquer site
- Veja o nome, preço e detalhes do produto automaticamente
- Organize sua lista de desejos de forma simples
- Visualize e acompanhe mudanças de preço

## Como funciona
1. Cole o link de um produto na sua wishlist
2. O app busca automaticamente as informações principais do produto (nome, preço, imagem, etc.)
3. Acompanhe todos os seus desejos em um só lugar
